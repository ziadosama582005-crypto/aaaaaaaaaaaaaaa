# 🛍️ نظام المتجر الإلكتروني المتقدم

نظام متجر رقمي متكامل مع نظام إدارة شامل بناءً على Firebase.

## 🎯 الميزات الرئيسية

### ✅ نظام المستخدمين
- المصادقة عبر OTP (بريد إلكتروني/SMS)
- المحفظة الرقمية مع عمليات الشراء
- تاريخ الطلبات والمشتريات
- إدارة الرصيد والمدفوعات

### 📦 نظام المنتجات
- إنشاء وتعديل وحذف المنتجات
- تصنيفات منتجات مرنة
- أنواع تسليم (فوري/يدوي)
- تتبع المبيعات للمنتج

### 🛒 نظام السلة والشراء
- إضافة المنتجات للسلة
- حفظ السلة في Firestore
- عملية شراء متقدمة
- تسجيل المشتريات مع البيانات المخفية

### 👁️ صفحة المشتريات
- عرض جميع المشتريات
- تفاصيل الطلب في modal
- نسخ البيانات المخفية
- إمكانية المشاركة

### 💳 نظام الدفع المتقدم
- محفظة رقمية
- الدفع بالبطاقة
- التحويل البنكي
- إدارة الرصيد

### 📊 لوحة التحكم
- إحصائيات شاملة
- إدارة المنتجات
- إدارة الطلبات
- إدارة المستخدمين
- التقارير المتقدمة

### 🔐 ميزات الأمان
- بدون Telegram Bot (كما طلب المستخدم)
- OTP للمصادقة
- تشفير البيانات الحساسة
- التحقق من جلسة المستخدم

## 🔄 تدفق العملية

### عملية الشراء
```
1. المستخدم يتصفح المنتجات
   ↓
2. يختار منتج → يضيفه للسلة
   ↓
3. يعرض السلة
   ↓
4. يتم الدفع (من المحفظة)
   ↓
5. تسجيل الشراء في Firestore
   ↓
6. عرض في صفحة "مشترياتي"
```

## 🎨 واجهات الاستخدام الجديدة

### صفحة السلة
- عرض جميع العناصر المضافة
- حذف العناصر من السلة
- ملخص السعر الكلي
- زر الشراء المباشر

### صفحة الفئة
- عرض المنتجات المتاحة
- عرض المنتجات المباعة
- عرض مشتريات المستخدم
- إضافة للسلة بسهولة

### صفحة تفاصيل المنتج
- عرض معلومات المنتج الكاملة
- نوع التسليم (فوري/يدوي)
- حقل لتفاصيل المشتري (للتسليم اليدوي)
- أزرار شراء مباشر/إضافة للسلة

### لوحة التحكم
- إحصائيات عامة (مستخدمين، منتجات، مبيعات، إيرادات)
- الطلبات الأخيرة
- المنتجات الأكثر مبيعاً

### إدارة المنتجات
- جدول بجميع المنتجات
- إضافة منتج جديد (Modal)
- تعديل المنتج
- حذف المنتج
- الحالة (متاح/مباع)

### إدارة الطلبات
- عرض جميع الطلبات
- تصفية حسب الحالة
- عرض تفاصيل الطلب في Modal
- معلومات المشتري والبيانات المخفية

### إدارة المستخدمين
- جدول بجميع المستخدمين
- معلومات العائد (الرصيد، الطلبات)
- عرض تفاصيل المستخدم

### التقارير
- رسم بياني للمبيعات اليومية
- إحصائيات عامة
- متوسط قيمة الطلب
- معدل التحويل

## 🔗 الـ API Endpoints

### السلة والشراء
- `POST /api/cart/add` - إضافة للسلة
- `GET /api/cart/view` - عرض السلة
- `POST /api/cart/remove` - حذف من السلة
- `POST /api/cart/checkout` - إتمام الشراء

### الفئات والمنتجات
- `GET /t/<category_id>` - عرض الفئة
- `GET /api/category/<category_id>` - API للفئة
- `GET /categories` - جميع الفئات

### الإدارة
- `GET /admin/dashboard` - لوحة التحكم
- `GET /admin/products` - إدارة المنتجات
- `GET /admin/orders` - إدارة الطلبات
- `GET /admin/users` - إدارة المستخدمين
- `GET /admin/reports` - التقارير
- `GET /admin/api/stats` - الإحصائيات
- `POST /admin/api/products/create` - إنشاء منتج
- `POST /admin/api/products/<id>/update` - تحديث منتج
- `POST /admin/api/products/<id>/delete` - حذف منتج

## 🚀 كيفية الاستخدام

### 1. المستخدم العادي
```
1. الذهاب إلى /categories
2. اختيار فئة
3. اختيار منتج
4. إضافة للسلة أو شراء مباشر
5. عرض السلة والشراء
6. الذهاب إلى /my_purchases لعرض المشتريات
```

### 2. المسؤول
```
1. الذهاب إلى /admin/dashboard للإحصائيات
2. إدارة المنتجات: /admin/products
3. عرض الطلبات: /admin/orders
4. إدارة المستخدمين: /admin/users
5. عرض التقارير: /admin/reports
```
